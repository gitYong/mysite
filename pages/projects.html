Cartoon Interpolater

Fall 2016

An animation software for 2-D images.

The goal of this project was to create a program that would allow the user to import an image file and provide the tools needed to animate the image. The 3 tools provided to the user are: cage, skeleton, and spline interpolation. Cage manipulation allows the user to distort the region within the cage. Skeleton transform allows the user to rotate joints and regions of the image that surround the joint, using skinning. Spline interpolation moves the surrounding according to an interpolated spline. The program was written in Python with Pygame for the interface, as well as Numpy and Python Image Library (PIL) for speed. Check out this demonstration. Source code here.

GT-112

Spring 2015

An Open-CV driven racing game.

This racing game allows the user to control the motion of a cartoon car using the motion of their hands and feet. The program was written in Python and uses the OpenCV module. OpenCV was used for color detection to determine the position of the user's hands and feet, which is converted into steering and acceleration input. The calibration feature, game design, and racing AI were all written from scratch. Check out this demonstration. Source code here.
