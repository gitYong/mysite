GT-112

An OpenCV driven racing game!

Developed for my term project for 15-112 at Carnegie Mellon University

This racing game allows the user to control the motion of a cartoon car using the motion of their hands and feet. The program was written in Python and uses the OpenCV module. OpenCV was used for color detection to determine the position of the user's hands and feet, which is converted into steering and acceleration input. The calibration feature, game design, and racing AI were all written from scratch. Check out this demonstration. Source code here.
